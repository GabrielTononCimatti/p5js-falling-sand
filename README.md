# Falling Sand

Simulação de areia caindo feita com [p5.js](https://p5js.org/).

## Acessar

https://gabrieltononcimatti.github.io/p5js-falling-sand/

## Controles

- **Arrastar o mouse** — criar partículas de areia
- **↓** — gravidade para baixo (padrão)
- **↑** — gravidade para cima
- **Areia / Fundo** — escolher cores via color picker ou HEX
- **Limpar** — limpar a tela
